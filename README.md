# Where/Wear Camo

Multiplayer hide-and-seek party game built on Kenney assets (CC0). Hiders are
chameleons: stand still and your colors adapt to whatever you're standing on
(grass, dirt, water, bushes) — only a subtle outline gives you away. Seekers
hunt with a paintball gun; hit hiders join the seekers mid-round.

## Maps

Four themed maps rotate randomly each round, all generated by the same layout
engine with different tile frames and foliage palettes: **JUNGLE**, **AUTUMN
WOODS**, **SNOWFIELD**, and **BADLANDS**. Chameleon blending samples the
*displayed* colors, so autumn bushes camouflage you orange and snowfields
camouflage you pale. Scenery has no collision — nothing snags players or NPCs.

## How to run

Do **not** double-click `index.html` — browsers block asset loading from `file://`
and you'll see a green placeholder grid instead of the game.

```
# Terminal 1 — static client
cd client
python -m http.server 8123

# Terminal 2 — game server (needed for ONLINE MODE; first time: npm install)
cd server
node server.js        # listens on :3000
```

Then open **http://localhost:8123**. OFFLINE MODE works without the game server.

**Online mode**: pick a name → CREATE ROOM (share the 5-letter code), QUICK
JOIN, or JOIN WITH CODE. The host presses START GAME. Matches run 3 rounds of
2:00 with role rotation; the server is fully authoritative (movement, phases,
hit validation with `shootHit` claims checked against server positions, NPC
brains, strobe sync) and never sends hider positions to seekers during grace.
Server tests: `cd server && node test-two-clients.js` (19 checks).

## Controls (current build — Step 2 solo test)

Start from the **main menu** → OFFLINE MODE.

Desktop:
- **WASD / arrows** — move · **mouse** — aim (crosshair) · **click** — fire paintball
- **Space** — snap-blend to the ground you're on (3s cooldown, must stand still)
- **Tab** — toggle Hider/Seeker (testing only) · **Esc** — pause menu

Mobile (or add `?touch=1` to the URL on desktop):
- **left joystick** — move · **FIRE/BLEND button** — shoot (seeker) / snap-blend (hider)
- **pause button** (top-right) — resume / settings / main menu

Mechanics: hiders' colors adapt to whatever they stand on (full flat-fill when
settled); every 10s a low heartbeat warns, then blended hiders strobe a stripe
pattern for 0.8s; paintballs fly over all foliage and leave persistent orange
splats where they land; a hit hider yells "FOUND!" and joins the seekers.

**The music is the radar**: Retro Beat plays at 0.8x speed. When a hider and a
seeker come within ~320px of each other the tempo swells to 1.0x, and inside
~160px it hits 1.2x — both sides feel the danger, nobody sees a marker. (Tempo
stays at 0.8x during grace so it can't leak hiding spots.)

**Chirp** (C key / CHIRP button, SEEK phase only): taunt audibly from your
position — everyone sees the ring, and the NPC seeker WILL come investigate.
Chirps and time-spent-hiding are tallied; the round-end scoreboard crowns the
Chirp Champion (chirps/min while hiding).

## Round structure (2:00 total)

- **GRACE (0:20)** — hiders scatter and hide. Seekers are frozen at spawn and
  see NO trace of hiders (no avatars, outlines, or blends) until grace ends.
- **SEEK (1:40)** — hiders render for seekers (blended or not). Round ends
  early if every hider is found; survivors win at the buzzer.

## NPCs (always present, offline and online)

- NPCs carry names from pools: offline uses clearly-bot names (BOT-FERN,
  CPU-ROCKY, ...); online draws realistic usernames (jake02, swampqueen, ...)
  so NPCs are indistinguishable from real players. There is deliberately NO
  visual marker distinguishing NPCs.
- **Offline roster**: YOU hiding + 6 NPC hiders + 3 NPC seekers — a hunter and
  the advanced stalker hunt from SEEK start; a second hunter stays at spawn and
  only wakes with 45 seconds left OR when fewer than 2 hiders remain.
- **Online rosters**: 2-4 humans → exactly 2 seekers (the stalker always, plus
  either one human on a 50% roll or a hunter NPC) and 6/5/4 NPC hiders as more
  humans join. 5-9 humans → the stalker plus 1-2 human seekers (two candidates
  roll 50%, minimum one) and 4 down to 2 NPC hiders. Everyone else hides.
- NPC seekers patrol zones, take potshots at suspicious bushes, snap-aim
  spottable hiders, and investigate chirps — hunters fire 15% more often and
  stalkers 25% more. Found NPC hiders convert into basic hunters. The chase
  rule applies to ALL hiders equally, human or NPC.
- NPC hiders pick round-start spots at least ~1000px from the seeker spawn;
  if a human hider camps on top of one for 3-5 seconds it repositions at least
  300px away. Nobody can be infected outside the SEEK phase — grace-period
  shots are harmless by rule, on both client and server.

## Deploying (single service — the server also serves the game)

`server/server.js` statically serves `client/` at `/` and Socket.io at
`/socket.io/`, honors `process.env.PORT`, and exposes `/healthz`. Deploy the
repo to any Node host with WebSocket support (Render, Railway, Fly.io, a VPS):
build `cd server && npm install`, start `cd server && node server.js`. Note:
Cloudflare Workers/Pages cannot run a Socket.io server — use Cloudflare as a
DNS/proxy layer in front of the Node host if you want their domain/CDN/DDoS.
- **Chase rule**: move, chirp, or get caught strobing within a seeker's notice
  radius (~280px, ~340px advanced) and it LOCKS ON — pursuing at near-player
  speed, firing every 0.9s, breaking only if you escape the extended radius
  (~450px / ~500px) or outlast 15 seconds. Blended and still = invisible to it.
- **Advanced NPC extras**: remembers where you escaped and sweeps that area for
  20s; its patrols drift toward hiders' general vicinity.
- Two NPC hiders cover-seek real bushes, blend on arrival, and relocate every
  25-45s until found or round end. Found hiders convert into extra NPC seekers.

Online lobbies will list "+3" for these NPCs; real player cap is 9 per room.

## Scoreboard

Round end shows the map name, CHIRP CHAMPIONS for hiders (chirps, hide time,
chirps/min) and a SEEKERS - PLAYERS FOUND table crediting each seeker's tags.
